Switch Display to DVI

Display settings can be modified from userspace using sysfs. This application demonstrates use of sysfs to modify framebuffer mode to 480p and switch display device from LCD to DVI. DVI display can also be enabled in the bootloader. Bootargument omapdss.def_disp="dvi" enables default boot display to DVI and framebuffer mode is defined using omapfb.mode=dvi:720x480@60.
